## 1. 如何访问

### 1.1 企业微信

1. 打开企业微信 → **工作台**
2. 找到 **IT 助手（IntelliServe）** 应用
3. 点击进入 → 开始对话

### 1.2 钉钉

1. 打开钉钉 → **工作台**
2. 找到 **IT 助手（IntelliServe）** 机器人
3. 点击进入 → 开始对话

## 2. 常用自助命令

### 2.1 网络问题

| 你可以这样说 | AI 如何处理 |
|-------------|-----------|
| `电脑连不上网` | 诊断网络 → 自动刷新 DNS / 重置网络配置 |
| `WiFi 连不上` | 检查无线网卡 → 复位网络适配器 |
| `VPN 连不上` | 诊断 VPN 客户端 → 提供配置检查步骤 |
| `网速很慢` | 检查带宽占用 → 提供排查建议 |

### 2.2 软件问题

| 你可以这样说 | AI 如何处理 |
|-------------|-----------|
| `Office 报错` | 识别错误信息 → 触发 Office 修复工具 |
| `软件打不开` | 诊断进程/依赖 → 建议修复或重装 |
| `电脑很卡` | 检查 CPU/内存 → 清理临时文件 |
| `蓝屏了，错误代码 0x...` | 识别错误代码 → 匹配知识库解决方案 |

### 2.3 硬件与外设

| 你可以这样说 | AI 如何处理 |
|-------------|-----------|
| `打印机连不上` | 重置打印服务 → 重新连接 |
| `U 盘识别不了` | 诊断 USB 驱动 → 提供修复步骤 |
| `显示器不亮` | 排查连接线/显卡 → 提供分步排查指南 |

### 2.4 账号与权限

| 你可以这样说 | AI 如何处理 |
|-------------|-----------|
| `密码忘了` | 引导自助重置或创建密码重置工单 |
| `无法访问共享文件夹` | 检查权限配置 → 创建权限申请工单 |

## 3. 截图提交

遇到复杂问题时，可以直接发送截图给 AI 助手：

1. **企微用户**：在聊天框点击 + → 相册 → 选择截图发送
2. **钉钉用户**：在聊天框点击 + → 图片 → 选择截图发送

AI 会提取截图中的文字信息（错误代码、报错内容），结合知识库进行诊断。

**截图技巧**：
- 尽量截取完整的错误窗口
- 包含错误代码和错误描述文字
- 如果是蓝屏，拍摄屏幕照片发送

## 4. AI 自助排障流程

```
你描述问题
    │
    ▼
AI 诊断 ──→ L1: 自动修复（无需你操作，等待几秒即可）
    │
    ├→ L2: AI 给出步骤引导，你按步骤操作
    │
    └→ L3: 自动创建工单，IT 工程师稍后联系你
```

**L1 自动修复示例**：
- 你：`电脑连不上网`
- AI：`已检测到 DNS 解析异常，正在为您自动修复...`
- AI：`修复完成！DNS 缓存已刷新，网络已恢复。请重新连接试试。`

**L2 步骤引导示例**：
- 你：`打印机连不上`
- AI：`请按以下步骤操作：1. 检查打印机电源是否开启 2. 检查 USB 线是否连接... 需要我帮您重置打印服务吗？[点击自动修复]`

## 5. 服务时间与人工升级

### 5.1 服务时间

| 时段 | AI 自助服务 | 人工支持 |
|------|-----------|---------|
| 工作日 9:00-18:00 | ✅ 7×24 可用 | ✅ 工程师在线 |
| 工作日 18:00-次日 9:00 | ✅ 7×24 可用 | ⚠️ 紧急工单响应 |
| 周末/节假日 | ✅ 7×24 可用 | ⚠️ 值班工程师响应 |

### 5.2 如何查看工单进度

1. 在 AI 助手对话中发送 `我的工单`
2. 或访问 Web 控制台：`https://<服务器 IP>/tickets`（用自己的账号登录）

### 5.3 紧急情况

如果你的设备完全无法使用（无法开机、无法联网），请直接通过其他方式联系 IT：
- 电话：IT 服务热线
- 或请同事代为创建工单，说明你的情况

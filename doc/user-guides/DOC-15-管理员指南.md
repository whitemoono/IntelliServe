## 1. 控制台概览

管理员登录后的首页仪表盘包含：

- **资产总览**：总数 / 在线 / 闲置 / 维修中，按类别饼图
- **告警摘要**：严重 / 警告 / 信息，点击跳转详情
- **工单趋势**：今日创建 / 已解决 / 待处理，近 7 天趋势折线
- **健康评分分布**：优秀 (>0.9) / 良好 (0.7-0.9) / 一般 (0.5-0.7) / 差 (<0.5)
- **AI 自动化统计**：L1 自动解决率、用户自助率

访问地址：`https://<服务器 IP>/dashboard`

## 2. 资产管理

### 2.1 手动录入资产

1. 导航至 **资产管理 → 新增资产**
2. 填写必填字段：名称、类别、SN 码
3. 可选：上传设备标签照片 → OCR 自动填充
4. 点击 **保存**

### 2.2 批量导入

1. 导航至 **资产管理 → 批量导入**
2. 下载 CSV 模板 → 填写 → 上传
3. 或批量上传设备标签照片 → OCR 批量识别
4. 预览 → 确认导入

### 2.3 查看与筛选

- 左侧分类筛选：台式机 / 笔记本 / 显示器 / 打印机 / 网络设备 / 外设 / 软件
- 状态筛选：使用中 / 闲置 / 维修中 / 已报废 / 预留
- 搜索：名称、SN 码、型号
- 高级筛选：保修到期 (<30天)、闲置超过 14 天、健康评分 <0.6

### 2.4 资产详情

点击资产行 → 详情页包含：

- 基本信息（可编辑）
- 健康记录趋势图
- 当前分配用户
- 关联工单历史
- 已安装软件清单
- 变更审计日志

### 2.5 闲置资产盘活

1. 导航至 **资产管理 → 闲置资产**
2. AI 自动识别低利用率设备（14 天未上线 或 CPU 使用率持续 <5%）
3. 勾选 → **发起调配** → 选择目标部门/用户
4. 系统生成调配工单

## 3. 用户与角色管理

### 3.1 角色权限

| 角色 | 权限范围 |
|------|---------|
| **admin（管理员）** | 全部功能：用户管理、资产增删改、系统配置、报表查看 |
| **engineer（工程师）** | 工单处理、脚本管理、知识库编辑、资产查看、监控查看 |
| **user（普通用户）** | 自助聊天机器人、查看自己工单、知识库搜索 |

### 3.2 新增用户

1. 导航至 **系统设置 → 用户管理 → 新增**
2. 填写：员工 ID、姓名、邮箱、部门、角色
3. 若角色为 engineer，设置技能标签（如 network, windows, printer）
4. 系统自动生成临时密码，用户首次登录需修改

### 3.3 工程师技能管理

- 每个工程师可设置 1-5 个技能标签
- AI 分配工单时自动匹配技能 → 实现智能派单
- 管理员可查看每位工程师的当前负载（待处理工单数）

## 4. 脚本审批与管理

### 4.1 脚本风险等级

| 等级 | 说明 | 审批要求 |
|------|------|---------|
| **Low（绿色）** | 只读或可逆操作 | 无需审批，自动执行 |
| **Medium（黄色）** | 配置变更 | 用户确认 |
| **High（红色）** | 系统级变更 | 管理员审批 |

### 4.2 审批流程

1. 工程师提交高风险脚本 → 状态变为 **待审批**
2. 管理员收到通知 → 审查脚本内容
3. 安全扫描自动运行（检查危险模式）
4. 管理员 **批准** / **拒绝** / **请求修改**
5. 批准后脚本变为 **可用**

## 5. 知识库管理

### 5.1 发布文章

1. 导航至 **知识库 → 新增文章**
2. 填写标题、分类、标签、Markdown 正文
3. 预览 → **发布**（触发自动分块嵌入）

### 5.2 AI 生成的草稿审核

- 系统自动从已解决工单提取知识 → 生成草稿文章
- 管理员/工程师审核草稿 → 编辑 → 发布

### 5.3 效果统计

- 每篇文章显示：浏览量、有用票数、无用票数
- 定期清理低质量文章（有用率 <50%）

## 6. 报表查看与导出

### 6.1 工单效率报表

- **位置**：报表中心 → 工单效率
- **内容**：工单总数、解决率、平均解决时长、L1 自动解决率、用户满意度
- **维度**：按类别 / 按工程师 / 按时间趋势
- **导出**：PDF / CSV

### 6.2 IT 成本报表

- **位置**：报表中心 → IT 成本
- **内容**：年度折旧汇总、许可续费预测、预算建议
- **导出**：PDF / Excel

### 6.3 合规报表

- **位置**：报表中心 → 合规
- **内容**：许可超用清单、即将到期许可、违规软件检测
- **导出**：PDF

核心场景

五大能力模块

每个模块都可以独立使用，也可以通过工单、知识库与自动化引擎形成闭环。

AI 自助服务台

将用户的报障描述、截图和历史知识库统一检索，自动给出处置建议或创建工单。

  • 支持企业微信、钉钉与 Web 会话入口。
  • RAG 检索知识库，减少重复咨询。
  • 对无法自助解决的问题自动升级工单。

自动化修复引擎

内置常见桌面运维脚本，按风险等级执行审批、确认和回滚记录。

  • 覆盖网络重置、打印机、Office、缓存清理等场景。
  • 通过 Celery 编排任务，记录执行状态和结果。
  • 高风险操作进入工程师审批。

资产全生命周期

从 OCR 录入、盘点、健康评估到闲置识别，统一维护资产状态。

  • 资产标签 OCR 识别和批量导入。
  • 终端健康趋势、保修与库存状态跟踪。
  • 软件许可和合规审计能力预留。

网络运维增强

自动发现拓扑，关联网络设备、端口、链路和异常告警。

  • SNMP/LLDP 拓扑发现和链路推断。
  • 设备配置备份和基础变更审计。
  • 异常流量、链路故障与工单联动。

安全保障

企业级安全，数据不出内网

完全本地部署，端到端加密，多层次安全防护

TLS 1.3 传输加密

全链路 HTTPS 加密传输，内部 CA 签发证书，Phase 3 升级 mTLS 双向认证。

JWT + RBAC 访问控制

三级角色权限体系 — admin / engineer / user，细粒度 API 级别权限管控。

AES-256-GCM 加密

敏感字段列级加密存储，许可证密钥、API 密钥、SNMP 社区串等均加密保护。

全量审计日志

所有 API 调用记录至审计日志，包含用户、操作、资源、时间、IP、结果。

脚本执行分级审批

低风险自动执行、中风险用户确认、高风险工程师审批，确保操作安全可控。

完全本地部署

所有 AI 推理、数据存储、业务逻辑均在本地完成，不依赖外部云 API，数据不出企业内网。
